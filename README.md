# Expense Tracker

Small full-stack expense tracker. Backend API + simple web UI.

## Stack

- Backend: Node + Express + TypeScript + SQLite (`better-sqlite3`) + zod
- Frontend: React + Vite + TypeScript + TanStack Query

## Structure

```
backend/    Express API (POST /expenses, GET /expenses)
frontend/   React UI (form, list, filters, total)
```
